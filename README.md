# run
a program that converts your natural language text into shell commands

powered by g4f

tested on windows

### current features
- powered by gpt4
- explain command
- auto-execute (specified in `%APPDATA%\run.config` or `~/.config/run.config`

### requirements
- g4f
- shellingham
- getch (for use in linux)

### planned features
- none yet
